Bienvenue dans le projet PhotoShare
Ce projet a été réalisé par 6 étudiants de l'Univerité Grenoble Alpes, en 2eme année de DUT Informatique. Ce projet a été réalisé pour l'entreprise PhotoWeb.

 Pour utiliser correctement notre projet :

  - disposer d'un serveur Apache (nous avons utilisé wamp) (http://www.wampserver.com/)
  Après l'installation, aller dans le dossier wamp64, puis www puis déplacer le dossier du projet
  Renommer le dossier "projetPhotoWeb"
  Il vous suffit ensuite de lancer le logiciel, puis de vous rendre sur le site via http://localhost/projetPhotoWeb/

  - pouvoir utiliser sqlite3

  - initialiser la BD avec init.sql

  De plus, nous avons essayé de mettre en place un serveur Raspberry Pi, mais nous n'avons pas réussi a parametrer correctement la base de donnée, seule la page
  principale s'affiche. Le résultat est visible via: http://http://photoshareproject.ddns.net/projetPhotoWeb/

  Nous remercions l'entreprise PhotoWeb qui nous a permis de réaliser ce projet, ainsi que les différents profs de l'IUT qui nous ont aidé tout au long de ce projet.
  Ce projet nous a permis d'acquérir une bonne expérience, ainsi que de nous mettre dans des conditions de travail plus professionnelle.
